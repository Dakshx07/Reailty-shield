## üíª Getting Started

Follow these simple steps to get a local copy of Reailty-shield up and running on your machine.

### Prerequisites

Ensure you have Node.js and npm (or yarn) installed.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Dakshx07/Reailty-shield.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Reailty-shield
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
4.  **Start the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

The application should now be running locally, typically accessible at `http://localhost:5173`.
